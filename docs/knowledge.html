Core Concepts

Critical Expert in the Loop

The AI assists; the human decides. Every transcription is a hypothesis that requires expert validation. This counters the "automation bias" where users over-trust AI output.

Categorical Confidence

Instead of misleading numeric scores (92.3%), we use three meaningful categories: confident, uncertain, and problematic.

Hybrid Validation

Combines deterministic rules (date formats, currency patterns) with LLM-as-judge analysis from four perspectives: paleographic, linguistic, structural, and domain knowledge.

Promptotyping

Iterative development through AI dialogue. Documentation before code. Early validation. Minimal, readable code. The project was developed using this methodology with Claude.
